- **Frontend**: `ReactJS`
- **Architeture**: `REST`
- **Database**: `MongoDB`

## RUNNING THE APPLICATION

### Backend

```sh
docker-compose up
```

### Frontend

- first install the project dependencies

```sh
npm install
```

- then start the app

```sh
npm start
```
